# AGENTS.md for DroneBridge Documentation

## Purpose

This repository contains the GitBook documentation for the DroneBridge ecosystem.

The documentation covers three related but distinct projects:

1. DroneBridge for Raspberry Pi
   This is the deprecated legacy project.
2. OpenSource DroneBridge for ESP32
   This is the current open source ESP32-based DroneBridge project.
3. Drone Light Show Edition (DLSE)
   This is a spinoff based on DroneBridge for ESP32.

When editing documentation, make sure content clearly matches the correct project. Do not assume that instructions, hardware support, features, or limitations are identical across all three variants.

## Target Audience

The target audience includes both hobbyists and professionals.

Readers may have very different levels of technical experience, but they all have basic knowledge about drones and how they work. Write documentation so it is approachable for less technical users without becoming vague or unhelpful for advanced users.

## Writing Guidance

- Prefer clear, direct language over jargon.
- Explain project-specific terms when they are first introduced.
- Keep steps practical and easy to follow.
- When a topic is advanced, briefly explain the why before the how.
- Call out project-specific differences explicitly, especially when readers could confuse Raspberry Pi, ESP32, and DLSE instructions.
- If content applies only to the deprecated Raspberry Pi project, label it clearly as legacy or deprecated.
