# License

More information on how to obtain licenses in the future. For now Drone Light Show Edition of DroneBridge for ESP32 is in a testing-only phase.

## License Types

The Drone Light Show Edition of DroneBridge for ESP32 (DLSE) knows the following license types:

* TRIAL
* EVALUATION
* ACTIVATED

All licenses work offline and do not require an internet connection. To request a license, an internet connection must be available.
